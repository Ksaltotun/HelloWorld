Perform and display the results of the following operations for these
properties :
· x += y - x++ * z ;
· z = -- x - y * 5 ;
· y /= x + 5 % z ;
· z = x++ + y * 5 ;
· x = y - x++ * z ;
Note: for each operation please use default values: x = 6, y = 15, and
z = 4, you don’t have possibility to reassign properties.
